Employee Management Web Application

A PHP and MySQL web application implementing a functional employee management system that connects a backend database with a simple web frontend to perform various HR features.

Features include:

Employee Directory
• Search employees by name and other attributes
• Filter employees by department, start date, location, etc.
• View profile cards and detailed employee information

Employee Data Management and Promotions
• Add new employees with full details
• Update existing employee records
• Apply promotions with user-specified pay increases

Birthday Cards
• Retrieve and display employees whose birthdays fall in the current calendar month

Contract Terminations and Audit Logging
• Process employee deletions and log termination information (including reason, date/time, deleting user ID, and employee details)
